# My Portfolio

Personal portfolio website

**[Live Demo](https://my-portfolio-six-fawn-11.vercel.app/)**

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Vite  
- **Tools & Others**: Git, Visual Studio Code  

## Installation & Local Setup

```bash
# Clone the repository (corrected URL)
git clone https://github.com/longnt3003/my-portfolio.git

# Navigate to the project directory
cd my-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
open http://localhost:5173 to preview

# Build for Production
npm run build

# Preview the production build:
npm run preview
```
